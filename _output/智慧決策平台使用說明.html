1 登入說明

1.2 輸入帳號密碼

帳號為員工工號，例如: 226178

預設密碼格式為”bes@工號”，例如: bes@226178

如果無法登入顯示帳號或密碼錯誤，請點擊忘記密碼進行驗證並修改密碼後重新登入。

1.3 首頁說明

# 使用Python

```{python}
# | label: fig-polar
# | fig-cap: "A line plot on a polar axis"

import numpy as np
import matplotlib.pyplot as plt

r = np.arange(0, 2, 0.01)
theta = 2 * np.pi * r
fig, ax = plt.subplots(
    subplot_kw={'projection': 'polar'}
)
ax.plot(theta, r)
ax.set_rticks([0.5, 1, 1.5, 2])
ax.grid(True)
plt.show()
```
